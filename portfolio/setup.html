ตั้งค่าการเชื่อมต่อฐานข้อมูล

Supabase — กรอกค่าโปรเจกต์ของคุณ แล้วทดสอบ/บันทึก

หาได้จาก Supabase → Project Settings → API Keys (ใช้ anon public เท่านั้น ห้ามใช้ secret/service key)

บันทึก (เบราว์เซอร์นี้) = เก็บใน localStorage ใช้ได้ทันทีเฉพาะเครื่อง/เบราว์เซอร์นี้ (เหมาะกับการเข้า admin)

ดาวน์โหลด config.js = เอาไฟล์ไปทับ portfolio/config.js แล้ว deploy เพื่อให้ผู้เข้าชมทุกคนเห็นข้อมูล
